Sobre o projeto

Inspirado no tradicional Jogo-da-velha, Tic Tac Toe Mania busca trazer a experiência clássica e simples da brincadeira que marcou (e ainda marca) gerações.

O projeto surgiu com o objetivo de utilizar meus conhecimentos básicos da linguagem JavaScript e testar minha lógica na resolução de problemas. Além disso, o projeto conta com o uso de HTML e CSS.

Foram utilizadass ferramentas como loop, forEach, object e mais. Para saber mais detalhes, consulte o README oficial do projeto, disponível no Github.
